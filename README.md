Staqlt YouTube-Notion Sync Engine
Features

Push to YouTube: Change status to 'Ready to Sync' in Notion to upload/update video.

Pull from YouTube: New videos uploaded manually are auto-added to the Notion calendar.

Thumbnail Sync: Handles high-res thumbnail updates via public URLs.

OAuth 2.0 Security: Uses encrypted credentials for API access.

Setup

Configure a Google Cloud Project with YouTube Data API v3 enabled.

Create an Internal Integration in Notion.

Import the youtube-to-notion-workflow.json into n8n.

Map your Notion Database ID to the workflow.
